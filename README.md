# MoneyManager
Ứng dụng quản lý chi tiêu

# Chức năng
- Thêm/sửa/xóa các khoản chi tiêu, thu nhập, chuyển khoản
- Thêm/sửa/xóa tài khoản
- Biểu đồ thống kê chi tiêu, thu nhập theo tháng, năm
- Tìm kiếm theo ghi chú

# Mô hình sử dụng

Room + LiveData + MVVM
